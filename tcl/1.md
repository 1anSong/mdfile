## 每一条tcl指令都是“命令+参数" 的格式,第一个单词看作命令，其他的单词看作是命令的参数
```
puts "Hello,World"
输出结果:Hello,World
```

## TCL解释器运用规则把命令分成一个个独立的单词,同时进行必要的置换。
TCL置换分为以下三类
1.变量值换$

- TCL解释器会将认为$后面为变量名,将变量置换为它的值
```
set a "snow"
puts $a
输出结果: snow
```
2.命令置换[]

- []是一个独立的语句
```
set a [expr 3+4]
put $a
输出结果:7
```
3.反斜杠置换\

- 换行符,空格,[,$等被TCL解释器当作特殊字符对待的字符,加上反斜杠后变成普通字符.
```
set X = 1 
set Y = 1.5
puts "[expr $X+$Y]"
输出结果:2.5
puts "\[expr $X+$Y]"
输出结果:[expr 1+1.5]
puts "\[expr \$X+\$Y]"
输出结果:[expr $X+$Y]

```
